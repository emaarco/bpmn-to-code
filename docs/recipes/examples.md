# Examples

Projects using bpmn-to-code in practice.

## easy-zeebe

A Zeebe-based process automation example showing how bpmn-to-code integrates into a typical Camunda 8 / Zeebe workflow project. easy-zeebe also ships with Claude Skills that scaffold worker implementations, REST controllers, and test boilerplate around the generated Process API — a practical example of using [deterministic code generation and AI skills as complementary layers](/overview/why).

[GitHub Repository](https://github.com/emaarco/easy-zeebe)

## engine-safari

A multi-engine comparison project that demonstrates bpmn-to-code across different process engines (Camunda 7, Zeebe, Operaton).

[GitHub Repository](https://github.com/emaarco/engine-safari)
